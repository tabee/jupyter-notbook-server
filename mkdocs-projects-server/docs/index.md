# MkDocs Projects Server

Willkommen beim MkDocs Projects Server!

## Übersicht

Dies ist ein leeres MkDocs-Projekt, das als Ausgangspunkt für Ihre Dokumentation dient.

## Erste Schritte

1. Bearbeiten Sie diese Datei, um Ihre Dokumentation zu beginnen
2. Fügen Sie weitere Markdown-Dateien im `docs/` Verzeichnis hinzu
3. Aktualisieren Sie `mkdocs.yml` mit der Navigationsstruktur

## Lokale Vorschau

Um eine lokale Vorschau zu starten:

```bash
mkdocs serve
```

Dann öffnen Sie Ihren Browser unter `http://127.0.0.1:8000`

## Dokumentation erstellen

Um die statische Website zu erstellen:

```bash
mkdocs build
```

Die generierte Website befindet sich im `site/` Verzeichnis.
